# Aula HTML

Repositório com meu aprendizado de HTML, feito de forma autodidata através de 
vídeos. Cada conteúdo estudado foi aplicado na prática logo em seguida, 
funcionando como uma espécie de "caderno vivo": a explicação teórica e o 
código funcional ficam juntos, no mesmo lugar.

## Como o projeto está organizado

O conteúdo está dividido em arquivos separados por tema, cada um explorando 
uma funcionalidade específica do HTML:

- **`index.html`** — página principal, contém toda a explicação escrita 
  sobre o que foi estudado (meta tags, tags de formatação, elementos de 
  citação, links, imagens, mídia, divs e HTML semântico), com os conceitos 
  documentados ao lado dos exemplos aplicados.
- **`hello.html`** — primeira página feita, um "Hello World" simples, 
  marcando o início do aprendizado.
- **`formulario.html`** — página dedicada a formulários, explorando 
  diferentes tipos de campo (texto, e-mail, senha, radio, checkbox, select, 
  textarea). É o arquivo mais indicado para visualizar a estrutura e o 
  padrão de construção usado nas páginas.
- **`audio.html`** — exemplos de inserção e controle de arquivos de áudio.
- **`video.html`** — exemplos de inserção e controle de arquivos de vídeo, 
  incluindo atributos como poster, autoplay e controles customizados.

## Conteúdos estudados
- Tags de formatação (negrito, itálico, sublinhado, marcação, etc.)
- Meta tags
- Elementos de citação (`abbr`, `cite`, `q`, `blockquote`, `bdo`)
- Links absolutos e relativos
- Imagens e boas práticas de uso (direitos autorais, bancos de imagem gratuitos)
- Áudio e vídeo incorporados
- Divs e introdução a HTML semântico (`header`, `nav`, `footer`, `section`)

## Sobre o aprendizado
Este projeto está sendo construído em paralelo ao curso Técnico em 
Desenvolvimento de Sistemas (SENAI), como estudo complementar por conta
