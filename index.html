O que é html?

HTML é uma linguagem de marcação utilizada para criar páginas web utilizando de diversos aspectos para deixar a interface bonita, como por exemplo as tags e meta tags.

O que são tags e meta tags?

Meta Tags

As meta tags são utilizadas para forncer informações para o próprio computador a respeito da forma como deve ser lida o processo que está sendo executado, por exemplo:

meta charset="UTF-8"
meta name="viewport" content="width=device-width, initial-scale=1.0"
meta name= "description" content="Este é o meu primeiro site"
meta name="keywords" content="sites de teste,app,html"
meta name="author" content="Gabriel Barbosa"

Estas Meta tags são tanto como descrições que o computador vai ler, como a descrição do site, as palavras chave para pesquisa e o autor como também itens para a formatação, como o padrão de escrita, a escala do tamanho das letras e etc

As tags para formatação são as responsaveis por dar estilo a escrita, trazendo uma diferença maior para o site, como por exemplo:

Tags de Imagem

As tags de imagem são o que trazem cor ou interatividade para o site, permitindo que tudo seja visualmente atraente.
Por mais que curioso, as tags de imagem nao precisam de um fechamento, podemos forma-las da seguinte maneira: 'img src="lalalala"'

Entretanto, nós precisamos definir atributos para a imagem, como o width, height e alt que comandam a largura(em pixels), altura(em pixels[vale ressaltar que caso apenas uma medida seja adicionada, a outra será ajustada automaticamente de acordo com o tamanho inicial para não deformar a imagem]) e um texto alternativo para pessoas com deficiência visual, sendo um texto que é lido por sintetizadores de voz. Desta forma: 'img src="lalalala" width="500" height="300" alt="Imagem do logo da linguagem de marcação h.t.m.l."'

Além disso, vale ressaltar que devemos usar nossas próprias imagens, pois caso usemos imagem do google ou utilizando uma url que não temos direito, podemos sofrer tanto com a falha delas no site, caso alguma seja tirada de ar, ou até com direitos autorais.
Para isso, devemos procurar imagens em locais que tenhamos acesso gratis, como por exemplo:

-Pexels
-Pixabay
-Unsplash
-Freepik

A partir disso, podemos adicionar essas imagens em nosso site, utilizando o caminho relativo, ou seja, salvando a imagem dentro de uma subpasta do site e buscando essa imagem dentro desta subpasta, ao invés de utilizarmos caminhos absolutos e imagens já prontas.
Por exemplo, eu tenho uma subpasta chamada "img" e dentro dela tenho uma imagem de um carro, com isso podemos utilizar a tag 'img src="img/carro.jpg"'(utilizando os atributos caso desejados) para adicionar a imagem do carro em nosso site(é importante que salvemos essas imagens diretamente na pasta img, para que elas não fiquem presas em alguma outra pasta do computador e você não consiga acha-las em sua subpasta, para isso, você pode procurar a opção em seu próprio navegador para que ao selecionar uma imagem ele pergunte em que local você deseja salvá-la.

Mapeamento de imagem

O mapeamento de imagem é uma forma de criar links dentro de uma imagem, ou seja, você pode criar um link para cada parte da imagem, como por exemplo, nós temos a imagem da bandeira do Brasil, e podemos criar um link/caminho a partir de cada local onde o usuario clicar, por exemplo, caso ele clique na cor amarela, abrimos o link do google, caso ele clique na verde, abrimos o link do youtube e assim por diante.
Para isso, utilizamos recursos como o imagemap para mapear a imagem colocando formas geometricas e destinos para cada uma, e após isso o próprio site gera um código html com as dimensões exatas de cada forma e local onde voce separou, formando uma tag map e img (vale ressaltar que as tags são adicionadas de acordo com o tamanho original da imagem, então caso você utilize o width e height, as coordenadas permanecem anteriores, então isso desfaz o mapeamento, por isso, é recomendado que você deixe a imagem no tamanho que vai usar antes de mapear.).

Formulários

Os formularios são feitos pra interação do site, como o echo e read do shellscript. Por isso, eles são muito importantes
Todos os formularios começam com a tag 'form' e dentro dela nós podemos adicionar os inputs e outputs, por exemplo, dentro da tag 'form'podemos adicionar outras tags que são como diferentes campos de entrada pro usuario digitar, por exemplo o campo de texto que é um 'input type="text".
Os principais campos e tags são:

'form' que é responsavel pelos formularios
'input' que cria a caixa de entrada
'label' é como um rotulo para a caixa
'select' é um campo diferente do input, é como uma caixa de seleção
      'option' tag para as opções de cada resposta selecionavel
            'value=""' é o valor que será adicionado para cada seleção possivel
            'disabled' que é usado para a opção padrao como "selecione uma cor" mas que não é selecionavel
            'selected' define uma opção como pré selecionada
'button' é um botão clicavel
'textarea' é um campo de texto maior e com seu tamanho manipulavel
      'rows' para definir a quantidade de linhas
      'cols' para definir a quantidade de colunas

Além disso também temos os atributos para adicionar dentro das tags, como:

'for="id_exemplo"' utilizado dentro da tag label que faz com que o rotulo externo direcione o usuario para a escrita.
'id="exemplo"' que é o identificador para o 'for'
'type="..."' que da o tipo do campo input
      -'text' para caixas de texto
      -'email' para so aceitar padroes de emails
      -'password' que deixa a escrita oculta como uma senha
      -'submit' que é o botão de enviar do formulario
      -'value' texto interior do botao de enviar
      -'radio' é a area com as bolinhas selecionaveis
            'label e id' que são o rotulo da bolinha e o identificador
            'name' é o nome das opções, o nome é como a o quadrado onde cada um vai, por exemplo, ao adicionar todos com o name "animal", voce so pode escolher uma opção para adicionar nesse quadrado.
            'value' que é o valor, ele é o nome de cada opção
      'checkbox' que é como uma lista de multipla escolha, segue a mesma ideia do radio
      (vale ressaltar que nesse caso os names tem que ser diferentes)
'placeholder' que é o rotulo que fica dentro da caixinha de escrita
'required' adicionado dentro do campo input que faz com que o campo tenha seu preenchimento obrigatorio

Listas

As listas servem pra organizar melhor as ideias ou os afazeres.

Listas não ordenadas

As listas não ordenadas são aquelas que não possuem uma ordem específica, ou seja, não importa a ordem em que os itens estão, eles são apenas uma lista de coisas. Para isso, utilizamos a tag 'ul(unerdered list)' para iniciar a lista e 'li' para cada item da lista.
Desta forma:
'ul'
'li' Item 1 '/li'
'li' Item 2 '/li'
'li' Item 3 '/li'
'/ul'

Resultando em:

  • Item 1
  • Item 2
  • Item 3

Listas ordenadas

As listas ordenadas são aquelas que possuem uma ordem específica, ou seja, a ordem dos itens importa. Para isso, utilizamos a tag 'ol(ordered list)' para iniciar a lista e 'li' para cada item da lista.
Desta forma:
'ol'
'li' Item 1 '/li'
'li' Item 2 '/li'
'li' Item 3 '/li'
'/ol'

Resultando em:

  1. Item 1
  2. Item 2
  3. Item 3

Video

O video é a forma de dar videos para paginas atraves de um link. Ele é muito semelhante a tag de audio, entao nós podemos tanto criar outro arquivo para o video, quanto adiciona-lo diretamente na pagina principal, seguindo os passos anteriores da tag de audio. A tag de video tem algumas diferenças como o tipo de video que é 'video/mp4(o mais comum)'. Utilizando a tag 'video' e dentro desta tag a tag 'source src="caminho do video"(seja relativo ou absoluto) type="tipo do video(geralmente video/mp4 quando o o arquivo é mp4, mas pode ser video/webm quando o arquivo é webm)"' e fechando a tag 'video'.
Desta forma:
'video controls width="100%" controlsList="nodownload"
     'source src="video/racionais.mp4" type="video/mp4"'
'/video'

Quando o site é carregado, o video fica com a tela preta até que seja iniciado, entretanto, podemos adicionar algumas tags para configurar o reprodutor, como:
-'autoplay' para reproduzir o video assim que a pagina for aberta
-'poster="caminho de alguma imagem" para adicionar uma thumbnail -'controlsList="nodownload nofullscreen"' para tirar a opção de download e tela cheia
-'width e height' para mudar o tamanho
-'controls' para que o controle de reprodução apareça
-'disablepictureinpicture' para tirar a opção de picture-in-picture

Todos estes atributos são adicionados na tag video, como: 'video controls width...'
Por fim, podemos abrir a pagina!

o html semantico surgiu pra trazer um significado para o codigo, por exemplo, ao inves de usarmos 'div class="topo" e corrermos o risco do desenvolvedor futuro nao entender o que é um topo, nós podemos usar a tag 'header' que corresponde justamente ao topo, fazendo com que o navegador e até algum desenvolvedor entenda que aquela area corresponde ao topo, mesmo que o nome da div seja algo aleatorio como "praia", a tag 'header' é correspondente ao topo.
